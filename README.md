# Snake-love

*A Lua snake game built using the LÃ–VE2D framework*

## How to Play

1. **Clone the repository:**

   ```bash
   git clone https://github.com/leonardo-luz/snake-love path/to/snake-love
   ```

2. **Run the game:** 

   ```bash
   love path/to/snake-love
   ```

## Controls

* **`h`**: Move left
* **`j`**: Move down
* **`k`**: Move up
* **`l`**: Move right
* **`u`**: Increase speed
* **`d`**: Decrease speed
* **`q` or `Esc`**: Quit
